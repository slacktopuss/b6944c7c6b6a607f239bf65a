Доступ к сетевой папке

Для подключения к сетевой папке нужно получить логин и пароль у системного администратора. Также вам потребуется скачать и установить на компьютер программу OpenVPN Connect.

Если у вас 32-битная система Windows — воспользуйтесь ссылкой:

Скачать OpenVPN Connect V3 (32-битная система)

Windows 10 и Windows 8.1
  1. Нажмите кнопку Пуск, а затем выберите Параметры системы
  2. Справа, в разделе Характеристики устройства, посмотрите, какой Тип системы указан.
Windows7
  1. Нажмите кнопку Пуск, щелкните правой кнопкой мыши Компьютер и выберите пункт Свойства
  2. В разделе Система, посмотрите, какой тип системы указан.

После установки OpenVPN Connect

  • Закройте окно How to import
  • В следующем окне нажмите AGREE


  • Закройте окно Updates, нажав OK
  • В строке URL введите адрес сервера и нажмите NEXT
  • https://vpn01.fastcom.pro 


  • При появлении ошибки Certificate error нажмите Accept


  • В строке Username введите свой логин, полученный у администратора
  • В строке Password введите (а лучше скопируйте, без пробелов и лишних символов) свой пароль
  • Следующие две строки — Profile Name и Port можно не трогать, оставьте как есть
  • Поставьте метки в двух чекбоксах внизу и нажмите IMPORT


  • При повторном появлении Certificate error нажмите Accept
  • Если все сделали правильно, откроется окно с надписью CONNECTED
  • Теперь программу можно закрыть или свернуть, она продолжит работать в фоне


  • Для быстрого доступа к файлам можно создать на рабочем столе ярлык (правой кнопкой мыши)


  • В открывшемся окне введите адрес сетевой папки
  • \\10.10.35.188\сетевая папка 
  • Нажмите Далее, чтобы продолжить


  • Теперь дайте ярлыку имя
  • В имени лучше прямо указать, что это удаленный (не локальный) доступ к папке


  • Созданный ярлык будет работать только при подключении OpenVPN Connect
  • OpenVPN Connect должен одновременно работать на сервере (в офисе) и на вашем компьютере


  • При пером открытии ярлыка Windows потребует авторизацию
  • Логин:  OLGA 
  • Пароль:  1505 
  • Не путать с логином и паролем от программы OpenVPN Connect!
